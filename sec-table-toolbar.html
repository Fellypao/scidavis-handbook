The Table Toolbar.

This toolbar allows a quick access to the commands of the Table menu used to modify a table.

Figure 6.5. The SciDAVis Table Toolbar


Table 6.5. Table toolbar commands.

Icon Command Key Description
Dimensions command   modify the number of rows and columns of the table
Add Column command   add a new column to the table
Statistics on Columns command   compute statistical parameters on selected columns
Statistics on Rows command   compute statistical parameters on selected row
