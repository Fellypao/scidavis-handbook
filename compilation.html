Compilation and Installation from sources

To compile SciDAVis from sources, download the latest source distribution from the SciDAVis page on Sourceforge. It comes as a tar.gz archive containing detailed building instructions. If you are interested in the current development, you can also obtain a snapshot from the SciDAVis GitHub repository.
