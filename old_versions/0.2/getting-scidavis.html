How to obtain SciDAVis

The SciDAVis home page can be found at http://sourceforge.net/project/scidavis. Updates and news can be found there.

SciDAVis is distributed as a package with sources which have to be compiled. Compiled packages suitable for most Linux distributions and Windows can also be obtained. If you want to build SciDAVis from sources, don't forget first to download and install the Qt, QwtPlot3D, Qwt, liborigin, zlib, muParser, and GSL libraries.
