How to obtain SciDAVis

The SciDAVis home page can be found at https://sourceforge.net/projects/scidavis/. Updates and news can be found there.

SciDAVis is distributed as a package with sources which have to be compiled. Compiled packages suitable for most Linux distributions and Windows can also be obtained. If you want to build SciDAVis from sources, don't forget first to download and install the Qt, QwtPlot3D, Qwt, zlib, muParser, and GSL libraries/dependencies. And also Python, SIP and PyQt if you want to enable Python scripting in SciDAVis.
